# ربات فال حافظ و تاروت

این ربات تلگرامی با استفاده از هوش مصنوعی OpenAI، فال حافظ و تاروت ارائه می‌دهد.

## نحوه نصب

1. کلون کردن ریپازیتوری
2. نصب وابستگی‌ها: `pip install -r requirements.txt`
3. ایجاد فایل `.env` و وارد کردن توکن‌ها
4. اجرای ربات: `python bot.py`

## استفاده

پس از اجرای ربات، در تلگرام دستور `/start` را ارسال کنید.
